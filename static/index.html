Welcome to FuncAAS

Staring at your belly button and wondering who really needs containers? Starting to think microservices have grown too large? Wish every function you needed for your next killer app was available with 92% uptime(2) just a simple jQuery.get() away?

Modularity

No longer is your application dependent on single-language-logic. Leveraging the latest in REST API design, Funcaas moves stdlib functionality out to the cloud where. Need a math function? Got it. Wondering if your array is an array? We can tell you.

Scalability

With traditional runtimes and standard library functions, you never really know how far you can scale. After all, do you know how many times a second math.floor() can be called, or what your OS will charge you per invocation? Nope, neither do we. But with all your functions in the cloud, we can tell you exactly how slow and expensive these calls can be!

Robustability

We've all been there. You call console.log() one too many times and your server crashes. Well, no more. With FuncAAS we can guarantee service levels up to and exceeding industry-standard n nines. Stop worrying about how you host your functions and jump on the FuncAAS bandwagon today!

Currently Deployed Functions

/is/int/n

This method returns {"result": True} if n is integer-ish, and {"result": False} otherwise.

/format

POST a JSON payload like {"string": "this is a list of {}, {}, and {}", "formatter", ["a", "b", "c"]} using Python 3-style .format() semantics and you'll receive a response like {"result": "this is a list of a, b, and c"}, or an HTTP 500 on error.
